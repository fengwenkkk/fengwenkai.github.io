华为eSDK eLTE解决方案是对eLTE宽带集群产品eAPP SDK接口进行标准化封装，为合作伙伴提供集群用户管理、群组管理、语音点呼、语音组呼、人工转接、缜密监听、环境监听、实时视频回传与分发、视频上墙、短彩信发送与接收以及集群终端GIS订阅与地理位置信息上报等管理调度功能。

V100R005C10SPC100
适配eDC610 V100R005C10SPC100产品
修改内容：GIS订阅新增ExpiredTime字段，配置变更上报添加派接组消息。
新增功能：批量去订阅用户，按类型获取调度台所管理的用户列表，获取终端能力信息，通过调度台普通组号获取调度台派接组信息，按类型获取调度台所管理的群组列表，获取当前调度台创建的所有多选组信息，
          订阅群组，迟后加入某群组群组会话，创建多选组，删除多选组，拒绝视频回传，对一组摄像头进行视频轮询，终止并挂断正在进行轮询的摄像头，发起多选组组呼或抢权，释放多选组话权，
		  针对群组或用户对象静音，控制当前通话音量，发送短数据，获取动态组成员列表，获取用户或者群组的扬声器或麦克风，调度台发送即时消息，调度台发送即时消息回执。
